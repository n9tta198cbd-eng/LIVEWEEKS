# 0→90 Calendar - Life Calendar Generator

Веб-приложение для создания обоев iPhone с календарем жизни, который обновляется каждый день.

## Структура проекта

```
src/
├── index.html          # Главная страница
├── script.js           # Основной JavaScript
├── style.css           # Стили
├── server.py           # Локальный сервер для тестирования
├── api/
│   └── generate.py     # Python API для генерации изображений
├── img/                # Изображения (логотипы, превью)
└── fonts/              # Шрифты Helvetica
```

## Установка и запуск

### Требования
- Python 3.7+
- Библиотеки Python: `Pillow` (PIL)

### Установка зависимостей

```bash
pip install Pillow
```

### Запуск локального сервера

1. Перейдите в папку `src`:
```bash
cd src
```

2. Запустите сервер:
```bash
python server.py
```

3. Откройте в браузере:
```
http://localhost:8000/index.html
```

## Использование

1. Выберите язык (ENG/RU) и цвет обоев (черный/белый)
2. Нажмите кнопку "CREATE WALLPAPER"
3. Введите дату рождения и настройки
4. Скопируйте URL из модального окна
5. Используйте этот URL в приложении Shortcuts на iPhone для автоматического обновления обоев

## API

### Генерация календаря жизни

```
GET /api/generate?type=life&birth=2009-08-01&lifes=90&w=1179&h=2556
```

Параметры:
- `type=life` - тип календаря (life/year/goal)
- `birth=YYYY-MM-DD` - дата рождения
- `lifes=90` - ожидаемая продолжительность жизни (в годах)
- `w=1179` - ширина изображения (опционально)
- `h=2556` - высота изображения (опционально)

### Другие типы календарей

**Годовой календарь:**
```
GET /api/generate?type=year&w=1179&h=2556
```

**Календарь цели:**
```
GET /api/generate?type=goal&goal=My%20Goal&start=2024-01-01&deadline=2024-12-31&w=1179&h=2556
```

## Деплой на Vercel

1. Убедитесь, что файл `vercel.json` находится в корне проекта
2. Файл `src/api/generate.py` автоматически работает как Vercel Serverless Function
3. Установите домен `mainllac.vercel.app` в настройках проекта Vercel:
   - Зайдите в Settings → Domains → Add Domain
   - Введите `mainllac.vercel.app`
4. Убедитесь, что `requirements.txt` содержит все зависимости (Pillow)
5. После деплоя API будет доступен по адресу: `https://mainllac.vercel.app/api/generate`
